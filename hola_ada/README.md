# 1. Hola Ada

Este es el módulo introductorio al lenguaje Ada.

## Conceptos Clave
- La estructura básica de un programa principal es un `procedure`.
- **`with`** y **`use`**: Se utilizan para importar y agilizar el uso de paquetes externos. `with Ada.Text_IO; use Ada.Text_IO;` nos da acceso a herramientas de entrada/salida.
- **`Put`**: Imprime texto en consola sin saltar a una nueva línea.
- **`Put_Line`**: Imprime texto en consola y luego salta a una nueva línea.

## Compilar y Ejecutar
Para compilar y correr este proyecto desde esta carpeta usa:
```bash
alr run
```
