# 8. Proyecto Final: Sistema de Gestión de Acceso

Este es el reto integrador del curso. Combina todos los elementos aprendidos:
- **Tipos de datos** y strings.
- **Entrada/Salida** con interacción con el usuario.
- **Subprogramas**: Procedimientos para menús e ingreso de datos, y Funciones para verificar booleanos.
- **Control de flujo**: `case` para el menú de roles, `if/else` para la validación de contraseñas.
- **Bucles**: `loop` y `while` para la gestión de sesiones recurrentes.

## Competencias logradas
Si fuiste capaz de seguir, compilar y ejecutar este reto, comprendes a la perfección las bases de la programación estructurada y la estricta y segura sintaxis de Ada usando Alire.
